Math formulas in HTML

This section explain some of the difficulties to render formulas using solely HTML and css. We impose a condition that the HTML and css should be used in a natural way avoiding absolute possitioning of elements and a deep control of the metrics of the fonts.

Baseline alignment

Formula must be propertly aligned by the base line.

The sum of x and y. The formula E=mc2.

Status. Formulas that are represented as simple text are aligned automatically.

Vertical layout

Vertical layout is needed for fractions and sumations.

The formula x=−x2 .

The formula ∞∑n=12n.

Status. The usage of Flexbox solves partially the problems.

Missing feature. The vertical-align style does not works as needed because numerical lenghts have to be specified.
